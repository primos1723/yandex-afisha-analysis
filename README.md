# yandex-afisha-analysis
Исследование поведения пользователей сервиса Яндекс Афиша.  Анализ факторов, влияющих на повторные заказы, сегментация аудитории и рекомендации для маркетинговой команды.

# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Исследование поведения пользователей сервиса Яндекс Афиша для выявления факторов, влияющих на повторные заказы. Цель — помочь маркетинговой команде лучше понимать аудиторию и оптимизировать маркетинговые бюджеты.

## Данные
- **Период анализа:** июнь-октябрь 2024
- **Объём данных:** 284 737 заказов, 21 449 уникальных пользователей
- **Источник:** PostgreSQL база данных data-analyst-afisha

## Ключевые выводы
1. **61.6%** пользователей возвращаются для повторных покупок
2. Оптимальный диапазон для удержания: **2-3 билета**, выручка **500-750 руб.**
3. Критический период для возврата — **первые 7-10 дней** после покупки
4. Desktop-пользователи лояльнее mobile (64.3% vs 61.0%)
5. Регионы с высоким возвратом: Лесостепной край (72.9%), Ягодиновская область (67.2%)

## Инструкция по воспроизведению
1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать окружение и установить зависимости: `pip install -r requirements.txt`
4. Создать файл `.env` по примеру `.env.example` с параметрами подключения к БД
5. Запустить Jupyter Notebook и выполнить все ячейки
